# Analysis notebooks

This repository contains code used to analyse the data in Teo, Lavrinienko et al. 2025:

Evaluating long-term stool preservation methods for maximizing the recovery of viable human fecal microbiota

## Requirements

QIIME 2 is required to run most of the commands described in the Jupyter Notebook. To install the latest version of QIIME 2 refer to [these](https://library.qiime2.org/quickstart/amplicon) instructions. For detailed session information and the list of required R packages for data processing and plotting in R refer to the accompanying R session info file.

### File explanations

 - Jupyter Notebook contains code to process raw data and reproduce the QIIME 2 analyses, including stats and files used for main figures
 - R Markdown file contains code to process QIIME 2 outputs and plot the data in R
 - R session info file contains information on the R environment and required packages
